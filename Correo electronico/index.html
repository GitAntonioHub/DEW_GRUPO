Definición de correo electrónico válido

Un correo electrónico válido es una dirección que cumple con ciertos requisitos de formato y existencia para ser reconocida en Internet:

Formato estándar: El correo debe tener el formato usuario@dominio.extensión, donde:

  • Usuario: Puede incluir letras, números y ciertos caracteres especiales.
  • Dominio: Identifica el proveedor de correo (como gmail o hotmail).
  • Extensión: Indica el tipo de dominio (.com, .net) o un código de país (.es).

Existencia del dominio: El dominio debe existir y estar configurado con registros MX para recibir correos.

Disponibilidad de la dirección: La cuenta de usuario en el dominio debe existir para poder recibir mensajes.

No estar bloqueada: La dirección no debe estar suspendida o restringida.

Para validar el formato básico, se puede utilizar una expresión regular que verifique los componentes de un correo electrónico:

  • Parte local: Letras, números, puntos, guiones y guiones bajos.
  • @: Carácter obligatorio.
  • Dominio: Comienza con letras o números y puede incluir guiones.
  • Extensión: De 2 a 6 caracteres alfabéticos (ejemplo: .com, .es).

Esta verificación solo valida el formato, no garantiza que el correo exista o sea entregable.
